    Introduction

    Welcome to the Buclist, a bucketlist API that lets you manage a bucket list.

    A bucket list is simply a collection of of experiences, achievements, goals and dreams that a person hopes to have or accomplish during their lifetime.

    With Buclist API, you can integrate bucketlist functionality into various devices and apps.

    The root of all API endpoints is https://buclist.herokuapp.com/api and all responses are json

    Registration

    POST /users

    curl -X POST "https://buclist.herokuapp.com/api/users?email=douglass@adams.com&password=p@ssw0rd"

    If your request is successful, you would automatically be logged in and given an authentication token:

  "email": "douglass@adams.com",

    To begin using the Buclist API, you need to register an account with the API

    Authentication

    Login

    POST /auth/login

    curl -X POST "https://buclist.herokuapp.com/api/auth/login?email=douglass@adams.com&password=p@ssw0rd"

    The above command returns JSON structured like this 

    {
  "email": "douglass@adams.com",
  "token": {
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyIjp7ImlkIjoyLCJlbWFpbCI6ImRvdWdsYXNzQGFkYW1zLmNvbSJ9fQ.v2X9aU-MF7QU_x3Km4EbzE135OMLnbKeAazNhQY7ZG4",
    "expires_at": "2016-11-07T13:39:17.009Z"
  }
}

    Login with the email and password used when signing up to get an authorisation token. The token is only valid for 2 hours. The token needs to be included in te header of every API request for bucketlists and items.

    Logout

    DELETE /auth/logout

    curl -X DELETE "https://buclist.herokuapp.com/api/auth/logout" -H "Authorization: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyIjp7ImlkIjozLCJlbWFpbCI6ImFub3RoZXJAZW1haWwuY29tIn19.F2VsF_BuKuiS6Rvg6GFTBCICgn47XFHrNFrDe6D04vA"

    Logout with your authorisation token. This will void the token.

    Bucketlists

    Get All Bucketlists

    GET /bucketlists

    curl -X GET "https://buclist.herokuapp.com/api/bucketlists" -H "Authorization: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyIjp7ImlkIjoyLCJlbWFpbCI6ImRvdWdsYXNzQGFkYW1zLmNvbSJ9fQ.v2X9aU-MF7QU_x3Km4EbzE135OMLnbKeAazNhQY7ZG4"

    The above command returns JSON structured like this:

    {
  "id": 1,
  "name": "Nihil vel placeat reprehenderit eius autem magni et.",
  "date_created": "2016-11-07T16:33:40.826Z",
  "date_modified": "2016-11-07T16:33:40.826Z",
  "created_by": 1,
  "items": [
    {
      "id": 1,
      "name": "Blanditiis dolor est qui pariatur natus.",
      "date_created": "2016-11-07T16:36:00.877Z",
      "date_modified": "2016-11-07T16:36:00.877Z",
      "done": false
    }
  ]
}

    This endpoint retrieves all Bucketlists.

    Query Parameters

    Parameter Default Description
    q Allows searching for bucketlists based on the name. If it is not present, all the bucketlists are returned
    page 1 If set to false, the result will include Bucketlists that have already been adopted.
    limit 20 If set to false, the result will include Bucketlists that have already been adopted. The maximum allowed limit is 100.

    The parameters can be combined.

    GET https://buclist.herokuapp.com/api/bucketlists?q=dream returns the first 20 bucketlists
    GET https://buclist.herokuapp.com/api/bucketlists?page=3 returns all bucketlists starting from the 61st result.
    GET https://buclist.herokuapp.com/api/bucketlists?limit=40 returns the first 40 bucketlists
    GET https://buclist.herokuapp.com/api/bucketlists?q=dream&page=2&limit=10 returns all bucketlists with ‘dream’ in the name, starting from the 11th result and showing 10 results
    Remember to include your authorisation token in the request header

    Create a Bucketlist

    POST /bucketlists/

    curl -X POST "https://buclist.herokuapp.com/api/bucketlists?name=Do%20Not%20Panic" -H "Authorization: eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJ1c2VyIjp7ImlkIjoyLCJlbWFpbCI6ImRvdWdsYXNzQGFkYW1zLmNvbSJ9fQ.v2X9aU-MF7QU_x3Km4EbzE135OMLnbKeAazNhQY7ZG4"

    The above command returns JSON structured like this:

    {
  "id": 1,
  "name": "Do Not Panic",
  "date_created": "2016-11-07T16:33:40.826Z",
  "date_modified": "2016-11-07T16:33:40.826Z",
  "created_by": 1,
  "items": []
}

    This endpoint creates a bucketlist with the name given.

    URL Parameters

    Parameter Description
    name The name of the bucketlist to be created
